How to connect to an IRC channel

Nov 17, 2018 in HOWTO
1 min read

The Internet Relay Chat (IRC) is a great way for free & easy communication.

Here are 5 steps to get started:

  1. Get an IRC chat program, e.g. XChat

  2. In the program connect to a server by typing a command like

    /server chat.freenode.net

  3. Join a channel by typing the command

    /join <channel>

    for example

    /join #bmrr

  4. Start chatting

  5. To find out more the command

    /help

    is there for you. So are these links:

    • Internet Relay Chat Help
    • irc cheat sheet · GitHub
    • #Beginner Website

PS: #bmrr is an IRC channel started by our research group. Feel free to join and connect with us.
